Introduction:
The Titanic dataset contains passenger information used to analyze factors affecting survival during the 1912 disaster. It helps identify patterns related to gender, class, and age.

Dataset Overview:
The dataset includes 891 passengers with variables such as age, sex, passenger class, and survival status. It contains both categorical and numerical data with some missing values.

Methodology:
Data cleaning and exploratory analysis were performed to examine relationships between survival and key variables like gender, age, and class.

Statistical Analysis:
Descriptive statistics and survival rate comparisons were conducted to identify significant factors influencing survival.

Conclusion:
Gender and passenger class were the strongest predictors of survival, with females and first-class passengers having higher survival rates.
